XMEN Fullstack Framework

View on GitHub

XMEN Framework allows you to rapidly develop while keeping your code structured.

Taking advantage of MongoDB allows for cross-platform document-oriented database support.

XMEN is Open Source and made by community developers.

Getting Started

Step 1:

Download and install MongoDB and Node.js

Step 2:

Clone the repo: 

git clone git@github.com:johntayl/xmen.git

Step 3:

Run NPM install 

npm install

Step 4:

Run the server! 

node server.js
